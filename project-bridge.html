Project Overview

Developed at Nanyang Polytechnic, this full-stack application enables safe interactions between generations. As Group Leader, I managed sprint planning and architectural decisions while leading the backend development.

Key Contributions

Profile Management

Implemented full CRUD operations and form validation. Designed the UI with specific accessibility considerations for senior users.

Friends System

Developed secure friend request workflows and privacy controls, restricting interactions to verified connections only.

OOP Architecture

Applied OOP principles to create maintainable Python classes for User, Profile, and Friends models with robust validation.

DB Design

Designed a normalized SQLite schema with foreign key relationships to support efficient profile and connection management.

Reflection & Learning

  • Integrating multiple team-developed modules highlighted the importance of early architectural planning and clear coordination across routes and databases.
  • Designing profile management for both youths and seniors strengthened my understanding of human-centric and accessible UI design.
  • Implementing full CRUD operations deepened my backend development skills, particularly in validation and error handling.
  • As Group Leader, I learned to balance technical execution with team communication, integration, and deadline management.
  • With more time, I would focus on performance optimisation and structured user testing to further refine the platform.
