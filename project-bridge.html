Intergenerational Bridge

Addressing the generational gap between youths and seniors in Singapore through a secure, accessible web platform.

Role

Group Leader

Timeline

Oct 2025 - Feb 2026

Team Size

5 Members

Tech Stack

Python, Flask, SQLite

Project Overview

Developed at Nanyang Polytechnic, this full-stack application enables safe interactions between generations. As Group Leader, I managed sprint planning and architectural decisions while leading the backend development.

Key Contributions

Profile Management

Implemented full CRUD operations and form validation. Designed the UI with specific accessibility considerations for senior users.

Friends System

Developed secure friend request workflows and privacy controls, restricting interactions to verified connections only.

OOP Architecture

Applied OOP principles to create maintainable Python classes for User, Profile, and Friends models with robust validation.

DB Design

Designed a normalized SQLite schema with foreign key relationships to support efficient profile and connection management.
